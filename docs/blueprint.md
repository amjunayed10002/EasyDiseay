# **App Name**: EasyDiseay Crop Doctor

## Core Features:

- Image Upload and Preview: Users can upload a crop image via drag-and-drop or file selection, with a clear preview of the uploaded image.
- AI Crop Disease Analysis Tool: An AI-powered tool analyzes the uploaded crop image, identifying potential diseases based on visual symptoms. This includes image recognition and classification.
- Generative Treatment Advice Tool: Based on the identified disease, a generative AI tool provides relevant treatment recommendations and preventive measures.
- Optional Crop Identification: Users can optionally input the specific crop type (e.g., Rice, Tomato, Wheat) to enhance the accuracy of the disease detection.
- Multilingual Support: Ability to toggle between English and Bengali for application text and results, mirroring the provided design.
- Interactive Disease Report Display: Present the detected disease(s) and their corresponding treatment advice in a clear, easy-to-understand format.

## Style Guidelines:

- Primary color: A vibrant, nature-inspired green (#3DA33D) to convey health and growth, used for main actions and branding.
- Background color: A subtle, desaturated green-tinted white (#EBF5EB) to provide a clean and airy canvas for content.
- Accent color: A soft, pale yellow-green (#CCE699) that provides a harmonious contrast and highlights key information or secondary interactive elements.
- Headline and body font: 'PT Sans', a humanist sans-serif for its modern look, readability, and friendly yet professional appeal, suitable for all textual content.
- Utilize clean, line-based icons that align with agricultural and diagnostic themes, consistently colored with the primary green for unity.
- Maintain a minimalist, centrally aligned layout with ample whitespace to ensure focus on the core interaction areas and content.
- Incorporate subtle hover effects on interactive elements and provide visual feedback animations for image uploads and analysis processing.
